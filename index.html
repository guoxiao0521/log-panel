📊 Log Panel 测试页面

一个用于嵌入式 Web 应用的可拖拽日志面板

控制面板

测试日志输出

🎯 功能特性

  • ✅ 拦截 console.log / warn / error 输出
  • ✅ 可拖拽的浮动面板
  • ✅ 可最小化/最大化
  • ✅ 带时间戳的日志记录
  • ✅ 不同类型日志颜色区分
  • ✅ 支持对象和数组格式化显示
  • ✅ 自动滚动到最新日志
  • ✅ 清空日志功能

📦 使用方法

// 浏览器引入 <script src="dist/log-panel.umd.js"></script> <script> LogPanel.init({ position: 'bottom-right', width: 400, height: 300 }); </script> // ES 模块引入 import LogPanel from 'log-panel'; LogPanel.init();
